# 🍔 Food Ordering Landing Page

A modern, fully responsive Food Ordering Landing Page designed with a McDonald's-inspired theme. This project focuses on utilizing CSS Grid and Flexbox for layout management, demonstrating clean and scalable frontend development practices.

## 🚀 Live Demo
[View Live Project](https://abolfazlmotallebzadehcs.github.io/grid-project/)

## 🛠️ Tech Stack & Features

*   **Core Technologies:** HTML5, CSS3.
*   **Layout Systems:** Expert use of **CSS Grid** and **Flexbox** to create a structured and adaptive design.
*   **Responsiveness:** Mobile-first approach with custom media queries to ensure compatibility across all device sizes.
*   **Design Architecture:** Utilizes the **BEM** (Block Element Modifier) methodology for clean, maintainable, and scalable CSS.
*   **Styling:** Custom font implementation using `@font-face` and a defined color variable system (`:root`).
*   **Interactive Elements:** CSS-only hamburger menu for mobile navigation and hover-effect enhancements on product cards.

## 🎨 Key Features

* **Responsive Navigation:** Includes a sidebar menu that activates on smaller screens using checkbox hacks for state management.
* **Dynamic Sections:**
* **Hero Section:** High-impact call-to-action layout.
* **Menu Section:** Grid-based product display with discount badges.
* **Sale/Offer Section:** Interactive cards with hover-state transformations.
* **App Promo:** Sleek promotional section for mobile application downloads.
* **Store Locator:** Search-focused section for store discovery.


* **Modern Aesthetics:** Focus on typography, spacing, and balanced layout proportions.
